📋 BOM Templates

A BOM (Bill of Materials) template defines the standard inputs per tonne for each product. When an operator opens a production run in BOM mode, these values pre-fill the form. All values can be overridden with actuals — variances outside tolerance trigger warnings.

✅ Configured — BOM is set up, product can use BOM mode in production entry ⚠️ Not configured — operator must use Manual mode until BOM is set up 🔄 By-product — no WIP material component, handling costs only
